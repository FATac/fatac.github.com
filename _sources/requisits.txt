Requisits
=========
Per tal de posar en marxa l'entorn del projecte Arts Combinatòries, cal
satisfer una sèrie de requisits previs.

Sistema operatiu
-----------------
Actualment, degut a limitacions tecnològiques, el entorn del projecte només
pot executar-se sobre màquines \*nix. En aquests documents suposarem una
màquina basada en una distribució Ubuntu Linux 11.04 (Natty).

Software
---------
 * GCC i eines de compilació
 * Python 2.6
 * Python Distribute (a.k.a setuptools)
 * Gawk
