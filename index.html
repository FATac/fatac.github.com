Welcome to Arts Combinatòries project documentation¶

This document gathers information concerning purposes, development, reusability and configuration of Fundació Antoni Tàpies (FAT) Arts Combinatòries project.

Source code¶

This is an open project whose code is fully-available at GitHub.

Development methodology and history¶

This is a PivotalTracker managed project, where development is divided into histories which are grouped by ordered iterations, following SCRUM agile development methodology.
